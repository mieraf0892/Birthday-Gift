# Audio Files

Place your MP3 files here with the following names:

1. `perfect.mp3` - Perfect by Ed Sheeran
2. `all-of-me.mp3` - All of Me by John Legend
3. `at-last.mp3` - At Last by Etta James
4. `thinking-out-loud.mp3` - Thinking Out Loud by Ed Sheeran
5. `cant-help-falling.mp3` - Can't Help Falling in Love by Elvis Presley
6. `thousand-years.mp3` - A Thousand Years by Christina Perri
7. `sunshine.mp3` - You Are the Sunshine of My Life by Stevie Wonder
8. `make-you-feel.mp3` - Make You Feel My Love by Adele

## How to Add Audio Files:

### Option 1: Download from YouTube (Free)
1. Go to a YouTube to MP3 converter like:
   - https://ytmp3.nu/
   - https://320ytmp3.com/
   - https://y2mate.com/

2. Search for each song on YouTube
3. Copy the YouTube URL
4. Paste it into the converter
5. Download the MP3 file
6. Rename it to match the names above
7. Place it in this folder (`public/audio/`)

### Option 2: Use Your Own Files
If you already have MP3 files:
1. Rename them to match the names above
2. Copy them to this folder (`public/audio/`)

### Option 3: Use Free Music Libraries
For royalty-free alternatives:
- https://pixabay.com/music/
- https://freemusicarchive.org/
- https://incompetech.com/

## Important Notes:
- Files must be in MP3 format
- Keep file sizes reasonable (under 10MB each)
- Make sure you have the right to use the music files
- The player will automatically detect and play these files
