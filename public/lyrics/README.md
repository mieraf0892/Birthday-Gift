# Song Lyrics

Add full lyrics for each song in separate text files.

## How to Add Lyrics:

1. **Find the lyrics** on sites like:
   - https://genius.com/
   - https://www.azlyrics.com/
   - https://www.lyrics.com/

2. **Create/Edit text files** with these names:
   - `perfect.txt`
   - `all-of-me.txt`
   - `at-last.txt`
   - `thinking-out-loud.txt`
   - `cant-help-falling.txt`
   - `thousand-years.txt`
   - `sunshine.txt`
   - `make-you-feel.txt`

3. **Copy the full lyrics** from the website and paste them into the corresponding file

4. **Save the file** in this folder (`public/lyrics/`)

## Example:
For "Perfect" by Ed Sheeran:
1. Go to https://genius.com/Ed-sheeran-perfect-lyrics
2. Copy all the lyrics
3. Paste into `perfect.txt`
4. Save

The app will automatically load and display the full lyrics when you click the Lyrics button!
